# Mailroom Madness

Charity donation reporting and donor 'Thank You' letter generating python package.

### Prerequisites

Python V 3.6/2.7 

### Installing

To install:

Clone repo
```
$ git clone https://github.com/hcodydibble/mailroom.git
```

Install package
```
$ python install setup.py
```

Run 
```
$ mailroom
```
